API Documentation

Our API allows developers to seamlessly leverage Sengine.io's REPL and hosting servers in their applications through simple http requests..

Execution API

Execute JavaScript, Ruby, and Python code in its optimal runtime engine. This may be used to power browser and mobile text editors, multiplayer code wars, blog tutorials, and more.

http://sengine.io/v1/api/execute
{
  "data": //code goes here as a string 
}

Hosting API

Instantly deploy and host static content for thirty minutes. This may be used to power web eccentric text editors, drag-and-drop html/css/js applications, or even to preview your own applications in a production environment.

http://sengine.io/v1/api/host
{
  "data": //code goes here as a string
}

Integration

Provided below are examples of API calls, or http requests, for different front end frameworks and libraries.

jQuery

Using $.ajax

$.ajax({
  type: "POST",
  url: "http://sengine.io/v1/api/execute",
  data: {
    "data": //code goes here as string
  }
});

AngularJS

Using $http provider in a custom factory.

app.factory('httpFactory', function ($http) {
  return {
    execute: execute,
    hostHTML: hostHTML
  }
  function execute(code) {
    return $http.post('http://sengine.io/v1/api/execute/execute', {"data": //code goes here as string});
  }
  function hostHTML(code) {
    return $http.post('http://sengine.io/v1/api/execute/host', {"data": //code goes here as string});
  }
});
